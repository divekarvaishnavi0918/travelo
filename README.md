# Travelo - Travel & Tourism Website

Travelo is a modern and responsive travel website built using HTML and CSS.  
The website showcases travel destinations, travel packages, and tourism information with a clean and attractive user interface.

## Features
- Responsive design for mobile and desktop
- Modern and clean UI
- Travel destinations section
- Easy navigation layout
- User-friendly design

## Technologies Used
- HTML5
- CSS3
- Responsive Web Design
